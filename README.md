# Beauty Salon Informative Website (Next.js + Tailwind)

This is a modern, responsive, informative website template for a beauty salon.
It includes: About, Services, Gallery, Testimonials, WhatsApp CTA, Social links, and Contact.

## Tech
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (subtle animations)
- Lucide Icons

## Quick start
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Configure content
Edit: `src/content/site.ts` (name, colors, services, testimonials, links, contact info).

## Deploy
- Push to GitHub
- Import on Vercel (recommended) and deploy
